My Journey into Cloud Computing

Watson Mwangi

October 2024

Introduction

It was around April 2024 that I first came across cloud computing and that is where interest quickly stirred me into action to explore this fast-growing sector. Since then, I have delved into the ever expanding universe of cloud technologies and learned important lessons about the core principles, growing services offerings, and best practices that are defining digital innovation for years to come.

What I Have Learnt

Cloud Computing Fundamentals

My journey began with understanding the core concepts of cloud computing. I explored:

  • Service Models: Differentiating between IaaS, PaaS, and SaaS.
  • Deployment Models: Examining public, private, hybrid, and community clouds.

AWS Cloud

Diving into Amazon Web Services (AWS) provided a robust foundation. Key takeaways include:

  • AWS Global Infrastructure: Understanding how regions and availability zones enhance performance and reliability.
  • Key Services: Familiarity with EC2, S3, RDS, and more, understanding their roles in application deployment.

Google Cloud Platform

Exploring Google Cloud Platform (GCP) introduced me to:

  • Compute and Storage: Deploying applications using Compute Engine and managing data with Cloud Storage and BigQuery.
  • Networking: Configuring Virtual Private Clouds (VPCs) and leveraging load balancers for enhanced performance.

Skills Acquired

During my course of study, I have learned many skills, which include:

  • Deploying simple web apps: I now have the ability to deploy simple web applications using cloud services, ensuring they are accessible and scalable through effective environment configuration and resource management.
  • Setting Up Databases: I am skilled in setting up and managing cloud databases, enabling efficient data storage and retrieval for my applications. This includes utilizing services like Amazon RDS and Google Cloud SQL.
  • Creating CI/CD Pipelines: I have developed the skills to create continuous integration and continuous deployment (CI/CD) pipelines using tools like AWS CodePipeline and Google Cloud Build. This allows for automated testing and deployment, significantly improving development efficiency.

Future Aspirations

As I continue my learning journey, I aspire to:

  • Attain professional certifications in AWS and GCP.
  • Combine my data communication network skills with my cloud computing expertise for enhanced solutions.
  • Explore advanced topics like machine learning and AI on cloud platforms.
  • Contribute to open-source projects to apply my skills in real-world scenarios.

As I conclude, cloud computing is more than just a trend; it represents the future of technology. I am excited to be part of this evolving landscape and look forward to leveraging my skills to create innovative solutions.
